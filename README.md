# ASCII Anything

**Free online ASCII art converter** — Transform any image into beautiful text-based art instantly. No upload required, everything happens in your browser.

🔗 **Live:** [asciianything.com](https://asciianything.com)

---

## Features

- **🎨 Multiple Character Palettes** — Standard, Blocks, Circles, Matrix, Simple, and Detailed
- **🌈 Color or Grayscale** — Export with full RGB color or classic monochrome
- **📐 Adjustable Resolution** — From 20 to 200 characters wide
- **🔄 Invert Mode** — Flip brightness mapping for different effects
- **💾 Multiple Export Formats** — Download as TXT, HTML, or PNG
- **🔒 100% Client-Side** — Your images never leave your browser

---

## How It Works

1. **Upload** any image (JPG, PNG, GIF, WebP, BMP)
2. **Convert** — We analyze pixel brightness and map each to an ASCII character
3. **Download** — Export as plain text, colored HTML, or PNG image

---

## Tech Stack

- Pure HTML/CSS/JavaScript
- No backend, no dependencies, no tracking
- Canvas API for image processing
- Works offline once loaded
